Algorithm Reference

FCFS — Non-preemptive. Processes execute in arrival order. Simple but can cause the convoy effect.
Round Robin — Preemptive. Each process gets a fixed time quantum. Ideal for time-sharing systems.
SJF — Non-preemptive. Picks the shortest burst from the ready queue. Optimal average wait for non-preemptive.
SRTF — Preemptive SJF. Preempts on every new arrival if shorter. Globally optimal average waiting time.

Metrics

  • Turnaround = Finish − Arrival
  • Waiting = Turnaround − Burst
  • Response = First CPU − Arrival
